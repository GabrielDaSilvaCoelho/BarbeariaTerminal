Para o projeto BarberPro, foi adotado o Bootstrap 5 como Design System base, complementado por um conjunto próprio de componentes reutilizáveis, como sidebar, cards, formulários e botões padronizados. A escolha foi motivada pela necessidade de rapidez na prototipação, consistência visual entre telas, responsividade nativa e facilidade de manutenção do front-end em HTML, CSS e JavaScript puro. O uso do sistema permitiu padronizar espaçamentos, tipografia, cores e estados visuais, garantindo melhor experiência do usuário e alinhamento com as boas práticas de UI/UX exigidas na atividade.
